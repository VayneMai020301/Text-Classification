# How to run 
* Clone this repository 
```bash
git clone https://github.com/VayneMai020301/Text-Classification.git
```
* Environment Configuration
```bash
pip install nltk
pip install scikit-learn
pip install json
pip install joblib
pip install pandas
```
# Training 
```python
python main.py
```
# Evaluation 
```python
python evaluation.py
```
# Inference
```python
python inference.py
```
